# 📱 Pixy App

Pixy is an Android application that integrates map-based features using Mapbox. It provides users with an interactive and smooth mapping experience.

---

## 🚀 Features

* 🗺️ Interactive maps powered by Mapbox
* 📍 Location tracking
* 🔍 Search and navigation (if implemented)
* ⚡ Smooth and responsive UI

---

## 🛠️ Tech Stack

* **Language:** Kotlin / Java
* **Framework:** Android SDK
* **Maps:** Mapbox SDK
* **IDE:** Android Studio

---

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/agarwalcharu831-cell/Pixy-app.git
   ```

2. Open the project in Android Studio

3. Add your Mapbox access token:

   * Create a `local.properties` file
   * Add:

     ```
     MAPBOX_ACCESS_TOKEN=your_token_here
     ```

4. Sync Gradle and run the app

---

## 🔐 Security Note

* Do **NOT** commit your Mapbox access token
* Always use `local.properties` or environment variables
